# Musix Lyrics

Project using plain JS that fetches lyrics using [musixmatch's API](https://developer.musixmatch.com/).

Technologies Used

- JavaScript (ES6+)
- TailwindCSS
- HTML
- CSS

Concepts Learned

- DOM Manipulation
- Asynchronous JS (async/await and promises)
- Fetch API
